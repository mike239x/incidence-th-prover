# class Triple

Represents triple of points. Has following methods:
 - `constructor`
 - `toString`
 - `fromString`
 - `clone`
 - `det`
 - `[Symbol.iterator]`

# function vector

Given points ABCDE returns a vector representing log[ABD] + log[BCE] - log[ABE] - log[BCE]
Uses VariablesManager for brackets indexation. Result may include log(-1).
